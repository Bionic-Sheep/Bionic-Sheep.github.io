二分法

前提：数组有序，无重复

重点：边界

在二分查找过程中，保持不变量，就是在while寻找中每一次边界的处理都要坚持根据区间的定义来操作。 —循环不变量

左闭右闭

定义target在左闭右闭的区间里，也就是**[left, right]**

· while(left <= right) 使用<=，因为left == right 有意义

· if(nums[mid] > target) right = mid - 1; 因为numd[mid] 一定不是target

左闭右开

target定义在左闭右开的区间，**[left, right)**

· while(left < right), left == right 在左闭右开的区间没有意义。

退出循环的条件：left == right。优点:不用判断left或right

· if(numd[mid] > target) right = mid; 因为当前nums[mid] 不等于target， 去左区间继续寻找，寻找区间为左闭右开区间，所以right = mid。

查找边界值

实现 int sqrt(int x) 函数。

计算并返回 x 的平方根，其中 x 是非负整数。

由于返回类型是整数，结果只保留整数的部分，小数部分将被舍去。

解：寻找X 的平方根就是查找k² <= X k的最大值。此时注意K是一个边界值。所以在while循环中丢掉了==这个阶段，将寻找答案融入到了< >中。
